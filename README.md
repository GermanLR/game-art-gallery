## 2. Краткое описание приложения
**Game Art Gallery** — мобильное приложение на Flutter, предназначенное для хранения, просмотра и простого редактирования пиксель-арта и игровых спрайтов. Пользователи могут загружать свои работы, систематизировать их по проектам и меткам, делиться ссылками и просматривать коллекции других авторов. Приложение решает проблему хаотичного хранения графики у начинающих пиксель-художников: вместо множества папок и разрозненных файлов — единая структура с метаданными и версионностью. Целевая аудитория — ученики, начинающие инди-разработчики, фан-артисты и все, кто создаёт небольшую игровую графику.

### Какие экраны представлены в макете
- **Home** — главный экран с лентой/сеткой работ и кнопкой создания/загрузки.
- **Details** — детальный просмотр работы: метаданные, комментарии, кнопки "Edit" и "Share".
- **Profile** — профиль пользователя с его коллекциями и настройками.


Минимальные UI-элементы, которые реализованы:
- кнопки (Create / Edit / Share / Back)
- текстовые поля (название работы, теги, описание)
- заголовки (экранов и карточек)


Формат: мобильный (вертикальная ориентация). Права доступа: ссылка на просмотр (Anyone with link — Can view).
